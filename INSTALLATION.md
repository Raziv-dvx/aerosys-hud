# AeroSys HUD Pro - Installation Guide

## Download & Installation

### ⚠️ Windows SmartScreen Warning

When you download and run the installer, Windows may display a security warning:

**"Windows protected your PC"** (blue screen with "Unknown publisher")

This is **normal** for unsigned applications. AeroSys HUD Pro is safe and open-source.

**To install:**
1. Click **"More info"**
2. Click **"Run anyway"**
3. Follow the installation wizard

### Installation Options

During installation, you can choose:
- ✅ **Installation Directory** - Choose where to install (default: `C:\Program Files\AeroSys HUD Pro`)
- ✅ **Desktop Shortcut** - Create a shortcut on your desktop
- ✅ **Start Menu Entry** - Add to Windows Start Menu (recommended)

### System Requirements

**Minimum:**
- Windows 10 or Windows 11
- Dual-core CPU (2 GHz+)
- 2 GB RAM
- 200 MB free disk space

**Recommended:**
- Windows 11 (latest updates)
- Quad-core CPU (3 GHz+)
- 4 GB RAM
- 500 MB free disk space

### First Launch

After installation:
1. Launch from Start Menu or Desktop shortcut
2. The app will appear in your system tray
3. Click the tray icon to open the dashboard

### Manual Taskbar Pinning

Windows doesn't allow apps to automatically pin to the taskbar. To pin manually:
1. Launch AeroSys HUD Pro
2. Right-click the taskbar icon
3. Select **"Pin to taskbar"**

## Features

AeroSys HUD Pro monitors:
- **CPU**: Load, temperature, per-core usage, throttling, power consumption
- **GPU**: Utilization, VRAM, temperatures, clock speeds, power draw
- **Memory**: RAM usage, channels, cache sizes, swap usage
- **Storage**: Disk usage, read/write speeds, SMART health status
- **Network**: Speed, ping, data usage (daily/monthly), Wi-Fi signal
- **Power**: Battery status, wear level, charge rate, system power draw
- **Additional**: Active window, audio volume, screen brightness, thermals

## Privacy & Data

- ✅ **All data stored locally** on your device
- ✅ **No telemetry** or tracking
- ✅ **No ads** or marketing
- ✅ **Open source** - view the code on GitHub

The only external network request is to ipify.org for public IP detection.

## Uninstallation

To uninstall:
1. Open **Settings** → **Apps** → **Installed apps**
2. Find **AeroSys HUD Pro**
3. Click **Uninstall**

All app data will be removed automatically.
